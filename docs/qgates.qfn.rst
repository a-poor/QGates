qgates.qfn
===========

The ``qgates.qfn`` includes helper functions for manipulating quantum gate and state matrices.

Specifically functions for creating state vectors, calculating the `tensor product <https://en.wikipedia.org/wiki/Tensor_product>`_, matrix multiplication, and conjugating a number or matrix.
