*created by Austin Poor*

A small library with some helper functions, quantum gates, and quantum states represented as `numpy <https://numpy.org>`_ matrices.

Originally written to help with the `Quantum Computing <http://science.slc.edu/~jmarshall/courses/fall/2019/quantum>`_ course I took at Sarah Lawrence College, taught by Professor James Marshall.

* GitHub: https://github.com/a-poor/QGates
* PyPi: https://pypi.org/project/qgates
* Read the Docs: https://qgates.readthedocs.io


Requirements
------------

* ``python>=3.6``
* ``numpy``


Installation
-------------
Installing with ``pip``:

.. code-block:: console

    $ pip install qgates
